# Easing reference

**Comet:** See `comet/tokens/transitions.json` for:
- cubic-bezier values (e.g. .65,.815,.735,.395; 0,0,.2,1)
- animation-duration (.3s, 1s, 1.5s)
- keyframes (slideIn, enter, shimmer, ping, indicator, indeterminate)
- ease-in-out, steps(48), steps(52)

**Atlas:** Motion framework — spring params, Bezier; see atlas/animation-system.md.
